# CaseAutoFramework-Web  向下翻有演示截图

## 后端传送门
[CaseAutoFramework](https://github.com/nightfall-w/CaseAutoFramework)

## 安装相关依赖包
```
npm install
```

### 开发环境的编译与运行
```
npm run serve
```

### 生产环境编译构建
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
